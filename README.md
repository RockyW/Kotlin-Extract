# Kotlin-Extract

Kotlin-Extract is a "toolbox" for Kotlin Android developer.  like[anko](https://github.com/Kotlin/anko):

>* SharedPreference
```java
// Any class
sp_saveString(...)
sp_getString(...)
sp_saveObject(...)
sp_getObject(...)
...

```

>* UI
```java

//in activity ,fragment ,view class
toast(...)
longToast(...)

dip2px(...)
sp2px(...)

loadColor(...)
loadDrawable(...)

```

>* Thread

```java
//Any class
async{
    ...
    runUI{
        ...
    }
}

```
